CSS Selectors and Specificity

Selectors

Type Selector

Targets all elements of a given type.

p { color: blue; }

Class Selector

Targets elements with a specific class.

.example { color: red; }

ID Selector

Targets an element with a specific ID.

#unique { color: green; }

Attribute Selector

Targets elements with a specific attribute.

[type="text"] { color: orange; }

Specificity

Specificity determines which CSS rule is applied by the browser when multiple rules could apply to the same element. It is calculated based on the types of selectors used:

  • Inline styles: Highest specificity (e.g., style="color: blue;")
  • ID selectors: Second highest (#id)
  • Class, attribute, and pseudo-class selectors: Next (.class, [attribute], :hover)
  • Type selectors and pseudo-elements: Lowest (element, ::before)

Example Calculation


Inline styles: 1000
ID selectors: 100
Class selectors: 10
Type selectors: 1

Example


<!DOCTYPE html>
<html>
<head>
<style>
  p { color: blue; } /* specificity: 1 */
  .example { color: red; } /* specificity: 10 */
  #unique { color: green; } /* specificity: 100 */
</style>
</head>
<body>
  <p class="example" id="unique">Text</p>
</body>
</html>

In this example, the text will be green because the ID selector has the highest specificity.
