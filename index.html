HTML: Elements

HTML elements are the basic building blocks of web pages. They consist of a start tag, content, and an end tag. 

<p>This is a paragraph.</p>

HTML: Structure

HTML documents have a structured layout including the following:

Tag Description
<!DOCTYPE> Defines the document type and HTML version
<html> Root element of an HTML document
<head> Contains meta-information about the document
<title> Sets the title of the document, shown in the browser's title bar or tab
<body> Contains the content of the document, such as text, images, and links 
<!DOCTYPE html>
<html>
  <head>
    <title>Document</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>

HTML: Tables

HTML tables organize data into rows and columns using the <table>, <tr>, <th>, and <td> tags. 

<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>

Semantic HTML

Semantic HTML introduces meaning to the web page rather than just presentation. Examples include <header>, <footer>, <article>, and <section>. 

<header>...</header>
<section>...</section>
<article>...</article>
<footer>...</footer>

Fundamentals of CSS

CSS (Cascading Style Sheets) describes how HTML elements are to be displayed. It controls layout, colors, fonts, and more. 

body {
  background-color: lightblue;
  font-family: Arial, sans-serif;
}

Selectors and Visual Rules

CSS selectors target HTML elements to apply styles. Specificity determines which styles are applied when multiple rules match the same element.

Selectors

Selector Example
Type Selector: p { color: blue; }
Class Selector: .example { color: red; }
ID Selector: #unique { color: green; }
Attribute Selector: [type="text"] { color: orange; }

Specificity Calculation

Inline styles: 1000
ID selectors: 100
Class selectors: 10
Type selectors: 1

The Box Model

The CSS box model describes the rectangular boxes generated for elements in the document tree and is made up of margins, borders, padding, and the actual content. 

div {
  width: 300px;
  padding: 10px;
  border: 5px solid black;
  margin: 20px;
}

The box model is represented by:

  • Content: The actual content of the box, where text and images appear.
  • Padding: Clears an area around the content. The padding is transparent.
  • Border: A border that goes around the padding and content.
  • Margin: Clears an area outside the border. The margin is transparent.

Display and Positioning

CSS offers several properties to control the layout and positioning of elements, such as display, position, float, and clear.

Display

key word Description
block The element is displayed as a block-level element.
inline The element is displayed as an inline-level element.
none The element is not displayed.

Positioning

key word Description
static Default position; elements are positioned according to the normal flow of the document.
relative Positioned relative to its normal position.
absolute Positioned relative to the nearest positioned ancestor.
fixed Positioned relative to the browser window. 
div {
  position: relative;
  top: 10px;
  left: 20px;
}

Colors in CSS

Colors in CSS can be specified using color names, HEX values, RGB values, RGBA values, HSL values, and HSLA values.

Color Names

body {
  color: red;
}

HEX Values

body {
  color: #ff0000;
}

RGB Values

body {
  color: rgb(255, 0, 0);
}

RGBA Values

body {
  color: rgba(255, 0, 0, 0.5);
}

HSL Values

body {
  color: hsl(0, 100%, 50%);
}

HSLA Values

body {
  color: hsla(0, 100%, 50%, 0.5);
}

Common CSS Properties

CSS properties define how elements are displayed. Here are some common properties:

  • color: Sets the text color
  • background-color: Sets the background color
  • margin: Sets the outer space around an element
  • padding: Sets the inner space within an element
  • font-size: Sets the size of the text
  • border: Sets the border around an element
